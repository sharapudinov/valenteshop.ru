Для изменения и доработки шаблона скопируйте шаблон и проводите изменения в нем, это необходимо для корректной работы обновлений решения. Для редактирования внешнего вида решения используйте файл стилей custom.css.Он подключается вместе с остальными стилями шаблона, но в отличии от них данный файл не затрагивают обновления.
